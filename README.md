# Image-tile-cropper
easy python tkinter croping


## Syntax of croping file:
```
#comment
name x0 y0 xL yL xN yN
# L - length - standard is 1, N - Number count - standard is 1
example1 0 0 100 100 2 4 
# image 100x100, 2 spans, 4 rows, start at [0,0]
```
